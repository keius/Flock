# API Endpoints

## HTML API

### Home

- `GET /` - loads React web app

## JSON API

### Users

- `POST /api/users`
- `PATCH /api/users/:userId`

### Session

- `POST /api/session`
- `DELETE /api/session`

### Groups
- `GET /api/group`
- `POST /api/group`
- `GET /api/group/:groupId`
- `PATCH /api/group/:groupId`
- `DELETE /api/group/:groupId`

### Events

- `GET /api/event`
- `POST /api/event`
- `GET /api/events/:eventId`
- `PATCH /api/events/:eventId`
- `DELETE /api/events/:eventId`
